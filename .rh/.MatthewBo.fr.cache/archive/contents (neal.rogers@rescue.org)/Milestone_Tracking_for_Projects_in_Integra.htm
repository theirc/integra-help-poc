Business Process Information Use of milestone project tracking is optional and may not be required in all locations/offices. This functionality can be used in place of Excel trackers for milestone results. Information stored in Integra can also be exported to Excel and copied/pasted into donor invoice templates. Below are examples of milestone markers that may be entered for a project: Peace building Research: Data collection underway Capacity Building: Preparatory work for capacity building training completed Database: Narrative summaries for subsets of literature drafted ‎ Set Up Milestone Tracking for a Project Purpose Use this procedure to set up milestone project tracking in Integra for Payment by Results projects. Prerequisites The D1 code (Project ID) exists in Integra. ‎ Steps From Integra home page, navigate to Project management and accounting > Project invoices > Award milestones. Click New on the top menu. Complete the following fields: Field Entry Required Company Enter the relevant legal entity code (if not already displayed). Due date Enter a due date for the milestone. Project ID Enter the Project ID/D1 code. Description Enter a description for the milestone. Financial value Enter an amount for the milestone. Milestone status Select the relevant milestone status from the drop-down list. E.g., New Update this status as milestone progresses. Payment status Select the relevant payment status from the drop-down list. Completed date Once milestone is completed, enter the completion date. Repeat steps 2 through 3 for each milestone. Click Save. To export the milestone tracking information to Excel, click the icon on the toolbar in the top right corner of screen and select Export to Excel. ‎ End of procedure Note: To update milestone statuses, display the same screen and make updates to the Milestone status field.
