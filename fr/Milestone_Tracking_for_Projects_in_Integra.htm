Aperçu des processus commerciaux

Informations sur le processus opérationnel

Le suivi des étapes principales des projets est facultative et n’est pas requise dans tous les lieux/bureaux. Cette fonctionnalité peut être utilisée à la place des fichiers de suivi Excel pour les résultats des étapes clés. Les informations stockées dans Integra peuvent aussi être exportées vers Excel et copiées/collées dans des modèles de factures donateur.

Voici des exemples de marqueurs d’étapes clés pouvant être saisis pour un projet :

Recherches sur la construction de la paix : collecte de données en cours

Renforcement des capacités : travail préparatoire à la formation de renforcement des capacités suivie

Base de données : résumés narratifs de sous-ensembles de projets de textes

Configurer le suivi des étapes clés d’un projet

Objectif

Utilisez cette procédure pour configurer le suivi des étapes clés d’un projet dans Integra pour les projets payés au rendement.

Conditions préalables

Le code D1 (ID de projet) existe dans Integra.
‎

Étapes

  1. Sur la page d’accueil d’Integra, accédez à Gestion et comptabilité du projet > Factures projet > Étapes clés de subvention.

  1. Cliquez sur Nouveau dans le menu en haut.

  1. Complétez les champs suivants :

Champ

Saisie requise

Entreprise

Saisissez le code pertinent de l’entité juridique (s’il n’est pas déjà affiché).

Date d’échéance

Saisissez une date d’échéance pour l’étape clé.

ID de projet

Saisissez l’ID du projet/code D1.

Description

Saisissez un description de l’étape clé.

Valeur financière

Saisissez un montant pour l’étape clé.

Statut de l’étape clé

Sélectionnez le statut pertinent de l’étape clé dans la liste du menu déroulant. P. ex., Nouveau

Mettez à jour ce statut à mesure que l’étape clé progresse.

Statut de paiement

Sélectionnez le statut de paiement pertinent dans la liste du menu déroulant.

Date de fin

Une fois l’étape clé terminée, saisissez la date de fin.
  1. Répétez les étapes 2 et 3 pour chaque étape clé.
  2. Cliquez sur Sauvegarder.
  3. Pour exporter les informations de suivi sur l’étape clé vers Excel, cliquez sur l’icône dans le barre d’outils en haut à droite de l’écran et sélectionnez Exporter vers Excel.

‎

Fin de la procédure

Note : pour mettre à jour les statuts des étapes clés, affichez le même écran et mettez à jour le champ Statut de l’étape clé.

Gestion et comptabilité de projet > Factures projet > Étapes clés de subvention.
