Schéma de processus

Lors de la réception de produits, le personnel doit gérer l’inventaire disponible.

Toutes les tâches sont gérées dans Integra, à l’exception de l’inventaire physique. Une fois le décompte physique terminé, le journal comptable est soumis et les résultats sont enregistrés dans Integra.

Le décompte physique peut être une fonction distincte de l’enregistrement et de la diffusion des résultats de décompte.

Créer un journal comptable - Décompte annuel

Objectif

Utilisez cette procédure pour créer un nouveau journal comptable pour un inventaire annuel dans Integra. Il est important de noter que, lors de la création d’un journal comptable pour un décompte annuel, le journal bloque l’inventaire dans le système et vous ne pouvez ni le prélever/emballer ni l’ajouter à un journal/une transaction différents tant que le journal comptable n’a pas été complété et diffusé.

Conditions préalables

n/a

Imprimer un journal de décompte

Objectif

Utilisez cette procédure pour générer une liste d'inventaire afin d’exécuter un inventaire. Cela peut être requis pour un décompte annuel, un décompte périodique ou un décompte de fin de période de subvention.

Conditions préalables

Le journal d'inventaire a été créé dans Integra.

Étapes

Dans la page d'accueil d'Integra, accédez aux modules Gestion des stocks  Écritures du journal  Décompte des articles  Décompte.

Dans la liste, sélectionnez le journal d’inventaire en cliquant sur le numéro de journal.

Cliquez sur  Liste d’inventaire.

 

Laissez l’option Imprimer disponible configurée sur Non.

Note : Concernant entrepôt, la meilleure pratique consiste à fournir une liste d'inventaire objective. Par conséquent, laissez l’option Imprimer disponible configurée sur Non afin que la fiche de décompte n’affecte pas le décompte avec les valeurs de décompte disponibles imprimées.

Dans la section Destination, la valeur par défaut est Écran. Pour imprimer l’écran, laissez la valeur par défaut.

Option : pour imprimer la fiche de décompte sur une imprimante, cliquez sur .

Cliquez sur .

Cliquez sur le menu déroulant dans le champ Nom et sélectionnez l’imprimante souhaitée.

Développez la section Registres à inclure et validez selon les besoins.

Cliquez sur  pour fermer le formulaire Liste d'inventaire.

Examinez les résultats de la liste d'inventaire.

Pour exporter la liste, cliquez sur Exporter et sélectionnez l’option souhaitée. P. ex., PDF, Excel, etc.

 

Cliquez sur pour imprimer la liste d’inventaire.

Cliquez sur X pour fermer l’aperçu de la liste d'inventaire.

Cliquez sur X pour fermer le Journal d'inventaire.

Résultat

Vous avez imprimé le journal comptable.

Imprimer plusieurs copies, selon les besoins.

Le personnel de l’entrepôt exécute l’inventaire dans l’entrepôt à l’aide du journal comptable imprimé.

Une fois terminé, le journal comptable est soumis au Superviseur.

Effectuer un inventaire physique

Objectif

Utilisez cette procédure pour réaliser un inventaire physique.

Conditions préalables

La liste d'inventaire a été générée et imprimée.

Étapes

Le personnel récupère une copie papier de la liste d'inventaire.

Le personnel compte physiquement tous les articles pour chaque numéro de ligne et enregistre l’inventaire dans la liste d'inventaire dans la colonne Maintenant décompté.

Le personnel envoie ou soumet la liste d'inventaire complète au personnel de la chaîne logistique sur place ou au responsable de l’entrepôt.

‎

Résultat

Vous avez réalisé l’inventaire physique et enregistré les résultats sur une liste d'inventaire sur papier.

Les résultats de l’inventaire peuvent maintenant être saisis dans Integra.

Enregistrer les résultats d’inventaire

Objectif

Utilisez cette procédure pour enregistrer l’inventaire physique dans Integra et comparer les résultats à l’inventaire disponible enregistré.

Conditions préalables

L’inventaire physique a été effectué et les résultats sont enregistrés sur la fiche d'inventaire.
‎

Étapes

Dans la page d'accueil d'Integra, accédez aux modules Gestion des stocks  Ecritures du journal  Inventaire  Décompte.

Dans la liste, sélectionnez le journal d’inventaire souhaité en cliquant sur le numéro de journal.

Cliquez sur  dans le menu en haut de la page.

En tant que superviseur, vous pouvez voir la colonne Disponible et ces valeurs sont renseignées.

Dans la colonne Décompté, saisissez le nombre physiquement décompté pour chaque article tel qu’enregistré dans le journal d’inventaire par le personnel de l’entrepôt.

Note : n’indiquez pas les valeurs dans le champ Quantité. Le système calcule la valeur pour cette colonne et la met à jour automatiquement.
Note : si la quantité est inférieure à la quantité disponible prévue, la colonne Quantité affiche une valeur négative. Si la quantité décomptée est supérieure à la quantité disponible, le champ Quantité affiche une valeur positive.

Écart entre les valeurs décomptées

En cas d’écart entre les valeurs disponibles et décomptées, la meilleure pratique consiste à effectuer une deuxième décompte. Le deuxième décompte est un processus manuel et a lieu hors du système et est réalisé par la même personne que celle ayant effectué le décompte initial. Le personnel de l’entrepôt soumet les deuxièmes valeurs décomptées au superviseur. Le superviseur saisit la deuxième valeur décomptée dans la colonne Décompté, et il s’agit du décompte définitif.

Dans la colonne Employé, pour chaque article décompté, saisissez le numéro d’employé de la personne ayant effectué le décompte.

Note : cliquez sur the menu déroulant et recherchez la personne à l’aide de son nom. Cliquez sur et son numéro d’employé sera renseigné dans le champ Employé.

Cliquez sur  une fois toutes les valeurs décomptées saisies.

Cliquez sur .

Vérifiez l’info-message indiquant que le journal d’inventaire a été validé avec succès.

Cliquez sur  dans la barre d’outils en haut de la page.

Résultat

Vous avez enregistré les résultats d’inventaire dans Integra.

S’il y a des écarts, vous les avez décomptés et effectué et saisi un deuxième inventaire.

Vous pouvez maintenant diffuser les résultats d’inventaire.

Ajuster une réservation d’inventaire de stocks pour traiter un ajustement

Objectif

L’inventaire est réservé pour un projet. Pour ajuster un solde d’inventaire à un article en stock, vous devez d’abord l’annuler pour un écart négatif (-) ou le réserver pour un écart positif (+).

Utilisez cette procédure pour annuler un article avec un écart négatif (-) ou réserver un article avec un écart positif (+) dans le projet pour lequel il est réservé.

Conditions préalables

Vous avez reçu un message d’erreur lors d’une transaction d’ajustement des stocks, tel que :

Étapes

Ouvrez une session dupliquée dans Integra.

Accédez aux modules Gestion de projet et comptabilité  Projets  Tous les projets.

Dans la grille Tous les projets et le champ Filtrer, saisissez l’ID de projet pour rechercher le projet auquel l’article est associé.

Cliquez sur à gauche de la ligne pour sélectionner le projet.

Dans le menu en haut, cliquez sur l’onglet Plan.

Dans la section Exigences concernant les articles, cliquez sur Exigences concernant les articles.

Effectuez un clic droit sur l’en-tête de la colonne et sélectionnez Insérer des colonnes…. pour ajouter la colonne Lieu à l’affichage.

Astuce : Cliquez sur  Sauvegarder afin de ne pas avoir besoin d’ajouter cette colonne à chaque fois.

Faites défiler la liste et Cliquez sur à gauche de Lieu – Aspects de l'inventaire.

Cliquez sur .

Note : l’ajout de la colonne Lieu vous permet de voir le lieu spécifique dans lequel l’inventaire devrait être annulé/réservé. Elle est ajoutée tout à droite de l’affichage en tant que dernière colonne.

Option : pour plus de commodité, cliquez sur l’en-tête de la colonne Lieu et faites-la glisser à droite de la colonne Numéro d’article.

Dans la liste des exigences concernant les article, trouvez le numéro d’article, le lieu et la date de réception demandée concernant l’article à annuler/réserver.

Cliquez sur à gauche de l’article pour l’annuler/le réserver.

Dans l’onglet Gérer, cliquez sur  Réservation.

Cliquez sur le champ Réservation et mettez à jour la valeur.

Note : pour annuler un article avec un écart négatif (-), mettez à jour la quantité en la réduisant du nombre d’articles à ajuster.

Par exemple, dans l’inventaire annuel, si la valeur décomptée est de 139,00 mais la valeur de la colonne disponible est de 140,00, réduisez la valeur de réservation de 1 pour obtenir 139,00.

Note : pour réserver un article avec un écart positif (+), mettez à jour la quantité en l’augmentant du nombre d’articles à ajuster une fois le journal d'inventaire diffusé.

Par exemple, dans l’inventaire annuel, si la valeur décomptée est de 141,00 mais la valeur de la colonne disponible est de 140,00, augmentez la valeur de réservation de 1 pour obtenir 141,00.

Une fois la valeur mise à jour dans le champ Réservation, le processus d’annulation/réservation est terminé.

Note : ce processus ne requiert pas de validation ou de sauvegarde.

Cliquez sur X pour fermer le formulaire Réservation et revenir à la grille Exigences concernant les articles.

Revenez à ou procédez à la transaction d’ajustement des stocks dans laquelle vous avez reçu le message d’erreur concernant l’inventaire.

Résultat

Vous avez annulé/reservé la quantité de l’article réservé pour un projet.

Créer un journal d’ajustement des stocks

Objectif

Utilisez cette procédure pour créer un journal d’ajustement afin d’enregistrer les écarts d’inventaire.

Les journaux d’ajustement des stocks permettent d’ajuster l’élimination de l’inventaire, y compris les articles expirés, endommagés et volés. Un ajustement permet également d’ajuster un inventaire trouvé mais ne faisant pas actuellement partie de l’inventaire disponible.

Note : ce type de journal n’est pas utilisé pour effectuer un ajustement découlant d’un cycle d’inventaire. Les ajustements de cycle d’inventaire sont effectués dans le journal de cycle d’inventaire.

Conditions préalables

Il a été déterminé qu’un ajustement des stocks était requis.

Étapes

Dans la page d'accueil d'Integra, accédez aux modules Gestion des stocks  Écritures du journal  Article  Ajustement des stocks.

Cliquez sur dans le menu en haut de la page.

Complétez les champs suivants :

Champ/Bouton

Saisie/Sélection requise

Section d’aperçu

Nom

La valeur par défaut est INV_ADJ.

Laissez la valeur par défaut pour un journal d’ajustement.

Description

Le champ de description indique par défaut la dernière description saisie.

Mettez à jour la description avec un identifiant correspondant à ce journal d'ajustement. (p. ex.) Raison + Journal d’ajustement des stocks

Site

Sélectionnez le site dans la liste de sélection du menu déroulant.

Entrepôt

Sélectionnez l’entrepôt dans la liste de sélection du menu déroulant.

Cliquez sur pour continuer.

Dans la section Lignes du journal, cliquez sur  pour saisir une nouvelle ligne dans le journal.

Champ/Bouton

Saisie/Sélection requise

Section de paramètres

Numéro d’article

Nom de produit

Indique par défaut le numéro d’article.

Quantité

Identifie le nombre ou la quantité de l’article ajusté. Si l’ajustement concerne un retrait ou une élimination, saisissez un symbole négatif (-).

Saisissez la quantité de l’ajustement.

Prix coûtant

Indique par défaut le numéro d’article.

Coût indiqué

Indique par défaut le numéro d’article.

Faites défiler jusqu’à la section Détails de la ligne.

Cliquez sur l’onglet Aspects de l'inventaire.

Cliquez sur l’onglet Aspects de l'inventaire et renseignez les champs suivants :

Champ/Bouton

Saisie/Sélection requise

ID de projet

Site

La valeur indiquée par défaut provient de l’en-tête mais peut être modifiée au niveau de la ligne, selon les besoins.

Cliquez sur l’onglet Aspects financiers.

Cliquez sur .
